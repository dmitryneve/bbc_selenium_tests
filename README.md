# Selenium Automation Framework

## Technology stack
- Java 17
- Selenium WebDriver 4.18
- TestNG
- WebDriverManager
- SLF4J (logging)


## How to run
1. Clone repository
2. Configure 'config.properties' (browser, baseUrl)
3. Run 'mvn clean test'
